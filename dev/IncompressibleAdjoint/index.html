IncompressibleAdjoint.compute_sensitivity — Method
compute_sensitivity(model::DiscreteModel, params::Dict{Symbol,Any}, uh0,ph0,ϕu0, ϕp0; objective_function=compute_drag)

From the solution of the primal flow uh0 ph0, and the adjoint flow ϕu0 ϕp0 it computes the senstivities according to the objective function. J2 contributions are splitted and the gradients computed individually to avoid numerical cancellation

IncompressibleAdjoint.obj_fun — Method
obj_fun(model::DiscreteModel, params::Dict{Symbol,Any}, uh,ph, fun)

Wrapper that evaluates fun the objective function. fun is a user-defined function. See eg. compute_drag, compute_lift functions. It return fitnessval,E. fitnessval is the function value that need to be minimized. E is a value that we want to monitor.
